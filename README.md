# SowSmith

Container repo for Purtera **Parser OS** tooling.

## Layout

| Path | Purpose |
|------|---------|
| **`parser-os/`** | Parser OS — evidence compiler (artifacts → atoms → graph → packets). Install with `pip install -e ".[dev]"` from that directory. PyPI-style name remains `purtera-evidence-mvp` in `pyproject.toml`. |

The former **`Bang/`** directory was renamed to **`parser-os/`** so the on-disk layout matches the product name.
